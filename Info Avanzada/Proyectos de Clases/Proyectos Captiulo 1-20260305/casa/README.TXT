Proyecto "casa"
Autores: Michael Kölling y David J. Barnes
Traducción: Maximiliano Andrés Eschoyez y Martín Federico Pelliza (2025)

Este proyecto es parte del materiral para el libro

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2016

Se utiliza en el capítulo 1.
    
Es un proyecto muy simple para demostrar las características de los objetos.

Este proyecto se utiliza en general luego de estudiar el proyecto "figuras".
Agrega la clase "Cuadro" que se utiliza para hacer un dibujo. Se Puede
utilizar para experimentar mediante la modificación del código fuente.
