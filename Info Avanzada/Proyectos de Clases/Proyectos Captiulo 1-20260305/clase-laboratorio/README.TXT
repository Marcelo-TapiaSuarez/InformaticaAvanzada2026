Proyecto "clase-laboratorio"
Autores: Michael Kölling y David J. Barnes
Traducción: Maximiliano Andrés Eschoyez y Martín Federico Pelliza (2025)

Este proyecto es parte del materiral para el libro

   Objects First with Java - A Practical Introduction using BlueJ
   Sixth edition
   David J. Barnes and Michael Kölling
   Pearson Education, 2016

Se utiliza en el capítulo 1.

El propósito de este proyecto es ilustrar la interacción entre objetos y
examinar código simple. Ver capítulos uno y tres del libro.

Para usar este proyecto, cree algunos objetos Estudiante y luego cree un
objeto ClaseLaboratorio. Agregue los estudiantes a la clase usando el
método "anotarEstudiante". Imprima la lista de clases.
